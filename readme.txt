1. Crear base de datos llamada coding_islands. 

2. Importar dentro de la base de datos recién creada el archivo coding_islands.sql ubicado en el siguiente directorio: proyecto\resources 

3. En el archivo class_conexion.php ubicado en el siguiente directorio: proyecto/modelo/utilidades/  
Modificar las siguientes variables adaptándolas a nuestro servidor. Por ejemplo: 
    private $DB_host = "localhost";  
    private $DB_user = "root"; 
    private $DB_pass = ""; 
    private $DB_name = "coding_islands"; 

4. Ubicar la carpeta Proyecto dentro de la raíz de nuestro servidor, en caso de utilizar XAMPP, esta debe ir dentro de la carpeta htdocs. 

5. La base de datos tiene un usuario de prueba ya creado, llamado lotup y contraseña lotup
